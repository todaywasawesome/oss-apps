# OSS Applications

This is a GitOps repo that acts as the source of truth for the open source team at Codefresh demoing GitOps apps. 

# Directory Structure

```bash
├── README.md
├── app-configs
│   ├── Production
│   ├── Staging
│   └── README.md
└── manifests
    ├── README.md
    └── application-name
        ├── base
        ├── production
        └── staging
```
